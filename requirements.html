参加要項

本ページは 「キスもろくにできねェの？」 Webオンリーイベントの参加ルール・提出仕様をまとめたものです。

1. 基本ルール

  • 本イベントは ONE PIECE・スモーカー×ロー（スモロ） を中心とした二次創作Webオンリーです。
  • 二次創作の性質上、原作・出版社とは無関係の非公式ファン企画です。
  • 各種法令・プラットフォーム規約・著作権に配慮した作品のみご参加ください。

2. サークル参加の条件・手順

条件

  • 募集期間：2025/08/21 〜 2025/10/31
  • 参加費：770円（ピクリエ利用料）
  • 一次・二次創作ともに可（公序良俗・各規約に反しない範囲）

手順

  1. 本ページの要項を確認
  2. 下記「参加申し込み」ボタンからピクリエのイベントページへ
  3. フォーム入力・決済（必要な場合）
  4. サークルページの作成・作品登録

提出物

  • サークル名／ペンネーム／連絡先
  • サークルカット（推奨 1200×800px / JPG, PNG）
  • 紹介文（全角 200〜600字目安）
  • 作品プレビュー画像・本文（必要に応じて外部ストア／SNSリンク可）

3. 作品仕様

推奨フォーマット

  • 画像：JPG / PNG（長辺 1600px 程度を目安）
  • テキスト：UTF-8 / 改行・段落の見やすさにご配慮ください
  • 外部リンク：BOOTH / Pixiv / X（旧Twitter）等は各規約に準拠

表現ガイドライン

  • 原作設定・読者配慮を尊重し、過度な暴力・差別表現は避けてください。
  • 年齢制限が必要な作品は、プラットフォームの年齢確認ルールに従ってください。
  • 未成年に相当する描写や違法・不適切な内容は登録できません。

4. 禁止事項

  • 著作権・商標権その他の権利を侵害する行為
  • 法律・各プラットフォーム規約に反する行為
  • 誹謗中傷、差別、なりすまし等の迷惑行為
  • 無断転載・無断販売などの不正行為

5. 免責事項・作品の取り下げ

  • 本イベントは非公式ファン企画です。原作者・出版社・権利者とは一切関係ありません。
  • 要項違反・権利侵害が疑われる場合、予告なく公開停止・修正依頼を行う場合があります。
  • トラブルが発生した場合、当事者間での解決を基本とし、主催は可能な範囲で対応します。
